# Business Analyst

Owns Epic Breakdown, Story Narrative, Functional Breakdown, PRD, validation system, and UAT-testable Stories.

Rules:
- Epics are business value units.
- Stories must be UAT-testable.
- Functional Breakdown outputs only the required function point table.
- Exclude technical implementation, UI design, permissions, security, logs, deployment, runtime, and database fields from BA sizing outputs.
