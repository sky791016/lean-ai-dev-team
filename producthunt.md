Hi Product Hunt! 👋 I'm Kai Shi (史凯), founder of the Lean AI Methodology.

I built this because I kept seeing the same problem: AI coding tools generate code,
but no one validates the business value first.

**Lean AI Dev Team** fixes this with 8 specialized agents that run in sequence:

1. 🧭 **Business Planner** — defines "why" before anything starts
2. 📊 **Product Manager** — produces an ROI model and KPI dashboard (not just requirements)
3. 📋 **Business Analyst** — user stories, As-Is→To-Be flows, acceptance criteria
4. 🏗️ **Technical Architect** — ADRs, API contracts, Clean Core design
5. ⚡ **Frontend + Backend + Data** — run in parallel, all honor the architect's contracts
6. ✅ **Compliance PM** — four-loop check, conflict detection, Definition of Done

The methodology is based on my **精益AI方法论 (Lean AI Methodology)** —
a framework for enterprise AI-native transformation used across 20+ organizations in China.

**What makes it different:**
- Every task produces a quantified ROI model, not just code
- Agents read existing code before writing anything — no blind stubs
- Four closed loops (Value, Data, Model, Operations) are checked on every task
- Works with any codebase, any language
